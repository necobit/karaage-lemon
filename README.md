# からあげレモン 🍋🍗

スマホ縦向きで遊ぶ、1台2人対戦のブラウザゲームです。
上側プレイヤーがレモン果汁を降らせ、下側プレイヤーがお皿でからあげを守ります。

**プレイはこちら → https://necobit.github.io/karaage-lemon/**

## 遊び方

スマホを縦向きにしてテーブルに置き、2人で画面を挟んで向かい合って遊びます。

### 攻撃側（🍋 レモン）
- 自分側（画面半分）のエリアを**タップ**すると、そこからレモンの水滴が落ちていきます
- 連打できますが、短いクールダウンがあります（画面上のレモンのリングが目安）
- 中央線ギリギリから落とすと相手が反応しにくい！

### 防御側（🍗 からあげ）
- 一番手前に並んだからあげの前で、**お皿を左右にドラッグ**して水滴をブロックします

### 得点ルール
- 水滴が1滴でもからあげに命中 → **攻撃側に1点**
- 10秒間守り切ったら → **防御側に1点**
- 得点が入るたびに**サイドチェンジ**（画面が180度回転して攻守交代）
- **先に5点取った方の勝ち！**

## モード
- **2人対戦** — 1台のスマホで向かい合って対戦
- **CPU対戦（攻撃側を操作）** — CPUの守るからあげにレモンを命中させろ
- **CPU対戦（防御側を操作）** — CPUのレモン攻撃からからあげを守れ

PCブラウザでもマウスのクリック / ドラッグで一応プレイできます（動作確認用）。

## バランス調整

主要なパラメータは `index.html` 冒頭の `CONFIG` 定数にまとまっています。

| 定数 | 意味 |
|------|------|
| `WIN_SCORE` | 勝利に必要な点数 |
| `DEFEND_TIME_MS` | 防御側が守り切る時間 |
| `TAP_COOLDOWN_MS` | 攻撃タップのクールダウン |
| `DROP_SPEED_RATIO` | 水滴の落下速度（画面高さ/秒） |
| `PADDLE_WIDTH_RATIO` | お皿の幅（画面幅比） |
| `KARAAGE_COUNT` | からあげの数 |
| `CPU_PADDLE_SPEED_RATIO` / `CPU_AIM_ERROR` | CPU防御の強さ |
| `CPU_ATTACK_MIN_MS` / `CPU_ATTACK_MAX_MS` | CPU攻撃の発射間隔 |

## 技術
- Vanilla JS + Canvas 2D、外部ライブラリなし、単一 `index.html`
- マルチタッチ対応（攻撃側・防御側の同時操作）
- WebAudio による簡易効果音
